# @starter/ui

Shared UI component library.

## Stack

- [shadcn/ui](https://ui.shadcn.com) - Component primitives
- [Base UI](https://base-ui.com) - Unstyled React components
- [Tailwind CSS v4](https://tailwindcss.com)
- [Lucide React](https://lucide.dev) - Icons
- [Recharts](https://recharts.org) - Charts
- [Vaul](https://vaul.emilkowal.ski) - Drawer
- [cmdk](https://cmdk.paco.me) - Command palette
- [Embla Carousel](https://www.embla-carousel.com) - Carousel
- [Sonner](https://sonner.emilkowal.ski) - Toasts

## Exports

```ts
import { Button } from "@starter/ui/components/button";
import { cn } from "@starter/ui/lib/utils";
import { useMobile } from "@starter/ui/hooks/use-mobile";
import "@starter/ui/globals.css";
```

## Adding Components

```bash
bun generate:components
```
